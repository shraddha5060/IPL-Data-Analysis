# IPL-Data-Analysis
IPL Data Analysis using Python and Pandas


## Project Overview
This project analyzes Indian Premier League (IPL) cricket data using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Most successful IPL teams
- Matches played per season
- Top Player of the Match winners
- Toss decision analysis
- Top run scorers
- Top wicket takers

## Dataset
IPL Matches and Deliveries dataset from Kaggle.

## Insights
- Mumbai Indians and Chennai Super Kings are among the most successful teams.
- Teams often choose to field after winning the toss.
- Certain batsmen consistently score the highest runs.
